# Compiler_Course_Assignments
The assignments for compiler course.

## Lexer
### 1. Install Meson
### 2. Clone this repo
``` bash
git clone https://github.com/Nowherechan/Compiler_Course_Assignments.git
```
### 3. Build with Meson
``` bash
cd Compiler_Course_Assignments/lexer
meson build
cd build
meson compile
```
